1132 法學德文

C 引註格式

ff.

✅ 「f.」= +1 條或頁，讀作「folgende」，如「Artikel 20 f.」= 「第 20 條及第 21 條」。 ✅ 「ff.」= 多條或多頁，讀作「fortfolgende」，如「Artikel 20 ff.」= 「第 20 條及其後多條」。 ✅ 條文唸法需按照「Artikel / Paragraf + 數字 + Absatz + Satz + Gesetz」的格式，確保準確性。 ✅ 在法律學術與法庭環境中，條文朗讀方式需符合標準，以避免誤解。

C.1.4 引註格式總結

德文法學引註採用以下邏輯順序：

  1. 法條引註（如 Art. XX GG）
  2. 評述語（如 zulässig: …）
  3. 專書／論文集條目（作者 + 書名 + 出版年 + 頁碼）
  4. 期刊文章（作者 + 簡寫期刊名 + 年份 + 頁碼）
  5. 使用縮寫如 ff. 表示「以下多頁」
